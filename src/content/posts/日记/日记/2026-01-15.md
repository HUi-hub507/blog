**今天有没有什么值得记录的事**

docker部署frp简直就是灾难，要不就是镜像太老了，喜欢用ini文件，要么就是格式问题。

下次记得固定格式

```
sudo docker run -d \
--restart=always \
--network=host \
--name "容器名字" \
-v /home/hui/frpc/"文件名字":/etc/frp/frpc.toml \
fatedier/frpc:v0.61.2 \
/usr/bin/frpc -c /etc/frp/frpc.toml
#示例
sudo docker run -d \
--restart=always \
--network=host \
--name frpc-poker \
-v /home/hui/frpc/frpc-poker.toml:/etc/frp/frpc.toml \
fatedier/frpc:v0.61.2 \
/usr/bin/frpc -c /etc/frp/frpc.toml
```

以后部署frp容器就用上面的格式，部署其他的容器还是写docker-compose.yml文件吧
